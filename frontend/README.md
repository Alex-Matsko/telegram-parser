# TG Price Monitor — Frontend

React frontend для системы агрегации цен из Telegram источников.

## Стек технологий

- **React 18** + TypeScript
- **Vite** — сборка
- **Tailwind CSS** — стилизация (dark theme)
- **Recharts** — графики истории цен
- **TanStack Table** — таблицы данных
- **TanStack Query** — управление запросами
- **React Router v6** — маршрутизация
- **Lucide React** — иконки

## Страницы

| Маршрут | Название | Описание |
|---------|----------|----------|
| `/` | Сводный прайс | Главная таблица цен с фильтрами, сортировкой, раскрытием строк |
| `/history/:productId` | История цен | График и таблица изменений цен по поставщикам |
| `/sources` | Источники | Управление источниками данных и сценариями ботов |
| `/unresolved` | Неразобранные | Сообщения для ручного разбора |

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение запускается на `http://localhost:5173` в режиме мок-данных.

## Сборка

```bash
npm run build
```

Результат — в папке `dist/`.

## Переменные окружения

| Переменная | Значение по умолчанию | Описание |
|------------|----------------------|----------|
| `VITE_USE_MOCKS` | `true` | Использовать мок-данные (установить `false` для работы с реальным API) |
| `VITE_API_BASE_URL` | `/api` | Базовый URL API |

## Docker

```bash
docker build -t tg-price-frontend .
docker run -p 3000:80 tg-price-frontend
```

Nginx проксирует `/api/` на `http://backend:8000/api/`.

## Мок-данные

В режиме `VITE_USE_MOCKS=true` приложение работает автономно с реалистичными данными:

- 32 товара Apple (iPhone, AirPods, Apple Watch, MacBook)
- 7 поставщиков
- 8 источников (каналы, группы, боты)
- 8 неразобранных сообщений
- 2 сценария ботов
- История цен за 1/3/7 дней
